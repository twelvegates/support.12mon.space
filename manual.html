Misskey設定マニュアル

質問する前にマニュアル見てね

基本設定

アイコンの変更

  1. 設定 → プロフィール → アイコン画像を変更
  2. アップロード → アイコンにする画像を選んで完了

プロフィールの変更

下記が変更可能なプロフィール情報です。

  • 名前
  • 自己紹介
  • 場所
  • 誕生日
  • 言語
  • 追加情報
  • 高度な設定
  • リアクションの受け入れ

※追加情報には、自分のサイトやSNSを入れることができる。設定項目は下記です。

  • ラベル: 任意の名前
  • 内容: URLなど

※高度な設定は下記2つの項目があります。

  • にゃああああああああああああああ！！！！！！！！！！ ※何が起こるかはご自身で体験ください。
  • Botとして設定: ※通常は設定しない

プライバシー設定

プライバシー設定では、公開したくないノートやつながり、オンラインの状態、フォローを承認制にするセッティングが出来ます。

特に設定を確認しておくべきところ

  • フォローを承認制にする
  • オンライン状態を表示
  • デフォルトの公開範囲

※デフォルトの公開範囲はパブリックで設定されています。

パブリックは当サーバーだけでなく、連合している数千のサーバーにも配信されますのでご注意ください。

リアクション

リアクションはTwitterにおける「いいね」と同じです。しかし、Misskeyではサーバー独自のカスタム絵文字を利用することができます。

設定 → リアクションではカスタム絵文字選択ポップアップの大きさ、列、高さを設定します。お好みで設定してください。

ドライブ

サーバーの各ユーザーには、割り当てられたドライブストレージの容量があります。過剰な画像・動画によりストレージがひっ迫するのを避けるため、300MBが上限に設定されています。 ユーザーには、ストレージ範囲内で画像・動画の投稿をしていただきます。

ドライブ設定では、下記のことが行えます。

  • アップロードした画像・動画ファイルの管理
  • 残容量のチェック
  • ドライブ内の不要なデータの削除

ドライブ内の不要なデータを削除するためには、ドライブクリーナーから行います。

通知設定

Misskey内のあらゆるアクションに対して、自分に通知を行うかどうかの設定ができます。

設定 → 通知 → 通知設定よりフォロー、メンションなどが発生した場合の挙動を設定します。

※プッシュ通知：PC・スマホのブラウザに対して、メッセージや音による通知を許可する設定です。

メール

詳しくはメール設定から設定方法を確認してください。

セキュリティ設定

ログインパスワード、authenticatorを使った二段階認証を設定できます。

また、ログイン履歴の確認もできます。

パスワードの変更

設定 → セキュリティ → パスワードから、パスワードを変更をクリック

※パスワードは確認入力がありません。急いで入力してEnter押すと詰むので、気を付けてください。

Google Authenticatorを使った二段階認証

詳しくはGoogle 認証システムで確認コードを取得するから設定方法を確認してください

クライアント設定

全般設定

下記の項目設定を行うことができます

  • UIの表示言語: Misskeyで使う言語
  • 表示デバイスタイプ
  • TL上への投稿フォーム表示
  • ノートの表示
  • 通知の表示設定
  • サーバー情報表示
  • センシティブなメディアの表示
  • 画像が1枚のノート表示
  • アピアランス(外観)表示
  • 絵文字のスタイル
  • フォントサイズ
  • サーバーとの接続が失われたときの通知表示
  • ページキャッシュ数
  • カスタムcss ※cssの知識がある方のみ

テーマ

テーマを変更したり、好きなテーマをダウンロードして自分の環境に適用する設定ができます。

テーマの変更・ダウンロード方法

  • プリインストールされているテーマは、プルダウンリストから選んですぐに適用できます。
  • テーマを探して、自分の環境に反映させることもできます。
    1. MiAS Theme Listで使いたいテーマのコードをCopy
    2. テーマのインストール
    3. Copyしたテーマコードを貼り付け
    4. インストール

ナビゲーションバー

ナビゲーションバーの構成、表示位置を設定します。(※デフォルトは一番左に表示)

ステータスバー

Federation(連合中のリモートサーバー)やRSSを流せる設定ができます。

追加 → タイプ - Fererationを設定すると下記のようになります。

サウンド

ノートが流れてきたとき、通知を受け取ったときのサウンドを自由に設定できます。

プラグイン

プラグインを入れることで処理の自動化を行ったり、Misskeyの通常の挙動を変更したりできます。

※JavaScriptやAPIに詳しくない方の操作は非推奨です。

その他の設定

ロール

自分に割り当てられたロールと権限が確認できます。

※管理者以外は基本的に変わらない権限設定をしています。

サーバーミュート

リモートサーバーの中でミュートしたいサーバーのURLを入力することで設定できます。サーバーとそのユーザーの全ての投稿がミュートになります

エンターキーで改行すれば複数のサーバーを設定できます。

ミュートとブロック

ミュート、Renoteのミュート、ブロックは該当ユーザーのページで行います

ワードミュート

指定ワードのノートをタイムラインに表示しないようにする設定です。下記2つの設定があります。

  • ソフトワードミュート: 指定したワードを含むノートをTLから隠す
  • ハードワードミュート: 指定したワードをTLから除外し、条件解除してもしばらくTL反映されない

API

Application Programming Interface。ソフトウェアにAPIという外部とやりとりする窓口を作り、 外部アプリとコミュニケーションや連携ができる状態にすること。サードパーティのmisskeyアプリとかで使います。

詳しくはスマホアプリを使うから設定方法を確認してください。

Webhook

詳しくはWebhook(Misskey公式)から設定方法を確認してください。

インポートとエクスポート

投稿したノート、フォロー、リスト、チャンネル、自分が行った各種設定値のインポート、エクスポートができます。

下記がインポート、エクスポート可能なデータの一覧です。

  • 全てのノート
  • お気に入りしたノート
  • フォロー
  • リスト
  • ミュート
  • ブロック
  • アンテナ

インポート・エクスポートはJSON形式です。

アカウントの移行

Misskeyではアカウントの移行をすることで、Twitterのように新規にアカウントを作り直したら今までのデータが飛ぶようなことはありません。

アカウントの移行方法は2つあります。

  1. 別のアカウントからこのアカウントに移行
  2. このアカウントを新しいアカウントへ移行

その他

アカウント情報の確認、アカウントの閉鎖が行えます。

(例) アカウントを閉鎖する
